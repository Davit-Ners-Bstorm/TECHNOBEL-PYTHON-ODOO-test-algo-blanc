# Tests blancs algorithmiques Python

## Duree recommandee

Chaque test est prevu pour durer **1h10**.

## Consignes

- Pas d'internet.
- Pas d'IA.
- Pas de correction avant la fin du test.
- Chaque exercice se lance directement avec :

```bash
python nom_du_fichier.py
```

## Objectif

Resoudre le plus d'exercices possible proprement.

Il vaut mieux proposer une solution claire et testee qu'une solution trop compliquee ou difficile a relire.

## Rappel

Pensez a la complexite :

- Combien de temps prend votre algorithme ?
- Combien de memoire utilise-t-il ?
- Peut-il gerer des entrees plus grandes ?
