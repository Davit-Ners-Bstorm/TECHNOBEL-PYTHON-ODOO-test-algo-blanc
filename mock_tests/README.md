# Tests blancs algorithmiques Python

## Duree recommandee

Chaque test est prevu pour durer **1h10**.

## Sessions disponibles

- `morning_test`
- `afternoon_test_2`
- `medium_low_bonus`

Pour `afternoon_test_2`, la duree recommandee est aussi de **1h10**.
La difficulte est legerement plus elevee sur l'exercice 5.

## Medium-low bonus

La session `medium_low_bonus` est optionnelle.

Duree recommandee : **1h10 a 1h30**.

L'exercice 5, `3Sum Bonus`, est plus difficile et peut etre garde pour la fin.

Ces exercices servent a entrainer des patterns plus avances :

- Prefix/Suffix
- HashMap + tri
- Two Pointers
- Prefix Sum + HashMap
- Tri + Two Pointers

## Consignes

- Pas d'internet.
- Pas d'IA.
- Pas d'anciennes corrections.
- Pas de correction avant la fin du test.
- Chaque exercice se lance directement avec :

```bash
python nom_du_fichier.py
```

## Objectif

Resoudre le plus d'exercices possible proprement.

Il vaut mieux proposer une solution claire et testee qu'une solution trop compliquee ou difficile a relire.

## Rappel

Pensez a la complexite :

- Combien de temps prend votre algorithme ?
- Combien de memoire utilise-t-il ?
- Peut-il gerer des entrees plus grandes ?
